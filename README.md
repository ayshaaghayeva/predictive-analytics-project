#  Bank Customer Churn Prediction

##  Overview
Develop a machine learning model to predict customer churn in a bank, enabling the identification of high-risk customers and supporting proactive retention strategies.

---

##  Workflow
- Data cleaning and validation  
- Exploratory data analysis (EDA)  
- Feature engineering  
- Encoding and scaling  
- Model training and evaluation  

---

##  Models
- Logistic Regression  
- Random Forest  
- XGBoost
